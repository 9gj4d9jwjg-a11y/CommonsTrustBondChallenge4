# XRPL Challenge 4: Cross-Chain Interoperability

## Challenge Overview
Trustbond.app is a blockchain application that helps NGOs raise donations transparently and prove their real-world actions by recording immutable evidence on the XRPL. This increases donor trust, using features like trustlines for issued currencies, on-chain evidence for accountability, and vaults (escrows) for conditional fund release.

Inspired by Interoperability (Day 2). Extend for cross-chain donations using Axelar SDK.

## What to Build
- Use Axelar for cross-chain transfers to XRPL.
- Integrate evidence/vaults cross-chain.
- Business: Map projects and analyze challenges (security/costs).

## Setup
1. Clone this repo: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Run: `node index.js` (Axelar requires API keys—get from https://axelar.network/).
